What this tool does

Most resume tools try to improve what's on the page. This one assumes the page is fine and asks whether you can actually defend it out loud.

How it works: Paste any text describing your experience. The tool extracts every specific claim — every metric, every verb, every outcome. Then it grills you on each claim, one question at a time. The sharper the claim, the sharper the follow-up.

Three sharpness levels: Friendly coach (warm probing), Balanced interviewer (fair grilling), Hostile grill (aggressive follow-ups — recommended for interview prep).

Backend status: If loaded inside Anthropic's Claude artifact environment, the tool uses live model calls for questions and scoring. Otherwise, it uses a rules-based skeptic that still asks specific, personalized questions from a library of 40+ challenge templates.

Your data: Everything stays in your browser. No uploads, no API key, no backend. Session history saves to localStorage.
